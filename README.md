# Todo App (React + TypeScript + Vite + Tailwind CSS)

Bu proje, **Web Geliştirme eğitiminde** istenen "TODO App" gereksinimlerini karşılayacak şekilde hazırlanmıştır. Amaç, modern bir frontend kütüphanesiyle (React) CRUD işlemlerini, component yapısını, styling'i ve basit bir state yönetimini (LocalStorage) uygulamaktır.

---

## 🚀 Bu Proje Ne Yapıyor?

- ✅ **Yeni görev ekleme** (Ekle)
- ✅ **Görevleri listeleme** (Listeleme)
- ✅ **Görevi düzenleme** (Güncelleme)
- ✅ **Görevi silme** (Silme)
- ✅ **Tamamlanan görevleri** ayırma
- ✅ **Bugün / Bekleyen / Gecikmiş** filtre sekmeleri
- ✅ **LocalStorage** ile tarayıcı kapatıldığında verinin korunması

---

## 📁 Proje Dosya Yapısı

- `src/components/` – Uygulama komponentleri (`AddTodo`, `TodoList`, `TodoItem`)
- `src/pages/` – Sayfalar (şu an sadece `Home` sayfası)
- `src/interfaces/` – Tip tanımlamaları (`Todo` modeli)
- `src/App.tsx` – Uygulama ana bileşeni
- `src/main.tsx` – React uygulamasının başlatıldığı yer

---

## 🛠️ Nasıl Çalıştırılır?

1. Bağımlılıkları yükle:

```bash
npm install
```

2. Geliştirme sunucusunu başlat:

```bash
npm run dev
```

3. Tarayıcıda aç:

```
http://localhost:5173
```
